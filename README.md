### 🎤 CampusSpeak AI

CampusSpeak AI is an AI-powered presentation practice tool that helps students improve their speaking confidence through real-time speech analysis and feedback.

Built with Flask, Web Speech API, and Gemini AI.

### ✨ Key Features

🎙 Live Speech Recognition – Real-time transcription using Web Speech API

🎯 Filler Word Detection – Detects words like um, uh, like

📊 Pace Analysis – Measures speaking speed (Words Per Minute)

💡 Clarity Score – Evaluates speech quality using heuristics

🤖 AI Coaching – Gemini AI gives personalized feedback

📁 Script Import – Supports .txt, .pdf, .docx, .ppt, .pptx

📝 Teleprompter Mode – Practice presentations with script tracking

🎨 Modern UI – Glassmorphism design with animated particle effects

### 🛠 Tech Stack

Backend: Flask (Python)
Frontend: HTML, CSS, JavaScript
Speech Recognition: Web Speech API
AI Feedback: Gemini API
Styling: Tailwind CSS

### 🚀 Run the Project

Install dependencies

pip install -r requirements.txt

Create .env

GEMINI_API_KEY=your_api_key
SECRET_KEY=your_secret_key

Run the app

python app.py

Open in browser

https://campus-speak-ai.vercel.app/
