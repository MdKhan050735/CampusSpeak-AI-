# 🌟 Demo Session - Quick Start Guide

## What is Demo Session?

A **clickable demo** that shows you exactly how the CampusSpeak AI analyzes presentations—without you having to record anything!

---

## 🚀 How to Use (30 seconds)

### Step 1: Open the App
- Go to the dashboard (either logged in or visit `/demo` for demo access)

### Step 2: Click "Try Demo" Button
- Look for the **purple "Try Demo"** button in the Practice Session area
- Located between "Start Practice" and "End Session" buttons
- Has a ⭐ star icon

### Step 3: Watch the Magic! ✨
- Demo loads instantly
- See a sample presentation script
- View real analysis metrics
- Read personalized coaching suggestions

## 🎨 Visual Features

### Filler Word Highlighting 🔴
Words like "uh", "like", "so" are highlighted in **bright red** so you can:
- See where fillers occur in the transcript
- Identify patterns in your own recordings
- Learn where to add pauses instead

### Metric Gauges 📈
Visual indicators show:
- **WPM Gauge** - Speaking pace with color coding (green = good, yellow = okay, red = needs work)
- **Clarity Gauge** - Clarity score visually represented
- **Progress Bar** - How much of the presentation is complete

### Analysis Modal 📋
Results displayed in a professional card showing:
- Top metrics (WPM, Clarity, Duration)
- Strengths you're doing well with
- Improvements to work on
- Personalized coaching tips

- Click "New Session" button to reset the interface

- No limits on how many times you can view

### Understanding the Metrics
| Metric | Ideal Range | What It Means |
|--------|-------------|---------------|
| WPM | 120-150 | Words spoken per minute - too fast or slow loses audience |
| Clarity | 80%+ | How clearly you're speaking - reduced filler words = higher clarity |
| Pace | Good/Bad | Overall assessment of speaking speed |
| Fillers | 0-2 | Minimize "um", "uh", "like", "you know", etc. |
